# Assets Directory

This directory contains images and decorative elements for the vintage newspaper aesthetic.

## Structure

### `images/vendors/`
Place vendor product images here. Images should ideally be:
- Black and white or sepia-toned
- High contrast for newspaper print aesthetic
- Square or portrait orientation

### `images/decorative/`
Place decorative elements here such as:
- Vintage ornaments
- Newspaper borders
- Decorative dividers
- Vintage seals/stamps

## Usage

Import images in components:
```javascript
import vendorImage from '../assets/images/vendors/vendor-name.jpg'
```
